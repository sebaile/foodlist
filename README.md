# 📖 About

**FoodList** is a web application designed to help users manage their food inventory, organize shopping lists, and plan meals more efficiently.

The platform allows users to keep track of products, monitor expiration dates, and create personalized shopping lists, making everyday kitchen management simpler and helping reduce food waste.

---

# ✨ Features

* 🥗 Food inventory management
* 🛒 Shopping list creation
* 📅 Expiration date tracking
* 🍳 Meal planning
* 🔍 Search and filter products
* 📊 Inventory overview

* ⚡ Fast and user-friendly interface

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/yourusername/FoodList.git
```

Enter the project directory

```bash
cd FoodList
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
python app.py
```

Open your browser

```text
http://127.0.0.1:5000
```
